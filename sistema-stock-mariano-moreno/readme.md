# Sistema de Gestión de Stock y Órdenes de Compra

Proyecto académico desarrollado para la materia Seminario de Práctica de la Licenciatura en Informática.

## Descripción

El proyecto consiste en una propuesta de sistema informático para la gestión de stock y órdenes de compra de la Escuela de Chefs Mariano Moreno.

El sistema busca mejorar el registro de ingredientes, el control de inventario, la generación de órdenes de compra y la trazabilidad de los movimientos de stock.

## Base de datos

Para la persistencia de los datos se utiliza una base de datos relacional MySQL.

La base de datos incluye las siguientes entidades principales:

- Usuario
- Ingrediente
- OrdenCompra
- DetalleOrdenCompra
- MovimientoStock

## Archivos incluidos

- `base_datos.sql`: contiene la creación de la base de datos y sus tablas.
- `consultas.sql`: contiene inserciones, consultas y borrado de registros.
- `comunicacion.md`: describe las definiciones de comunicación entre la aplicación y la base de datos.

## Tecnologías propuestas

- MySQL
- SQL
- Java
- JDBC
