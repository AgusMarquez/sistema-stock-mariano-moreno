# Sistema de Gestión de Stock y Órdenes de Compra

Proyecto académico desarrollado para la materia Seminario de Práctica de la Licenciatura en Informática.

## Descripción

Este prototipo en Java representa una versión inicial del sistema de gestión de stock y órdenes de compra para la Escuela de Chefs Mariano Moreno.

El programa permite administrar ingredientes, registrar ingresos y egresos de stock, detectar productos con bajo stock y generar órdenes de compra.

## Funcionalidades

- Agregar ingrediente general.
- Agregar ingrediente esencial.
- Listar inventario.
- Buscar ingrediente por nombre.
- Registrar ingreso de stock.
- Registrar egreso de stock.
- Ordenar inventario por nombre.
- Generar orden de compra por stock bajo.
- Listar órdenes de compra.
